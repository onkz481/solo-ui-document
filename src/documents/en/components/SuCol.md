##### Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| align-self | string | undefined | Sets the vertical alignment. |
| cols | number | undefined | Sets the number of columns in the component. Possible values are 1 to 12 or **auto**. |
